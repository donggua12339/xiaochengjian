# ADR 0043 · 商业化定价:订阅制

- 状态:accepted
- 日期:2026-07-14
- 层次:战略

## 背景

ADR 0002 定了"开发者订阅",需明确具体模式。

## 决策

**订阅制**

- **开源版**:免费(自部署)
- **SaaS 版**:月度订阅 + 终身会员 + VIP 解锁

## 备选方案

| 方案 | 免费 | 付费 | 不选原因 |
|---|---|---|---|
| 完全免费 | 全功能 | 无 | 无商业化 |
| 订阅制(本方案) | 开源版 | SaaS 月度/终身 | 合理 |
| 抽成制 | SaaS 免费 | 按卡密量抽成 | 开发者抵触 |
| 混合 | 开源+SaaS 基础 | SaaS 高级+按量 | MVP 太复杂 |

## 影响

- 正面:开发者一次性付费,无抽成压力
- 负面:无持续抽成收入
- 具体价格:见 ADR 0047
