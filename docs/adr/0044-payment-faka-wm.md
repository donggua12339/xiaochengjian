# ADR 0044 · 支付对接:发卡网 + 会员激活码

- 状态:accepted
- 日期:2026-07-14
- 层次:商业

## 背景

SaaS 收费需要支付通道。个人开发者无企业资质,官方支付门槛高。

## 决策

**发卡网(WM)+ 会员激活码模式**

- 用 WM 发卡网卖会员激活码
- 小城笺后台加"会员激活码兑换"接口
- 用户购买激活码后,在小城笺后台兑换会员

## 备选方案

| 方案 | 费率 | 资质 | 不选原因 |
|---|---|---|---|
| 微信/支付宝官方 | 0.6% | 企业 | 无企业资质 |
| 第三方聚合(Stripe) | 2-5% | 个人可 | 费率高 |
| 发卡网(本方案) | 0% | 无 | 合理 |
| 手动开通 | 0% | 无 | 无法规模化 |

## 影响

- 正面:0 费率,无需资质,开发者熟悉(卖卡密就是他们的业务)
- 负面:WM 不稳定时需手动兜底(ADR 0051)
- 开发量:会员激活码兑换接口约 2 天

## 关联

- 关联 ADR:0043(订阅制)、0051(WM 兜底)
