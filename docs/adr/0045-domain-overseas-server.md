# ADR 0045 · 域名 + 备案:国内域名 + 海外服务器

- 状态:accepted
- 日期:2026-07-14
- 层次:运维

## 背景

SaaS 面向国内开发者,但备案需要国内服务器 + 20 工作日审核。

## 决策

**国内域名 + 海外服务器(无需备案)**

- 域名:国内注册(.com)
- 服务器:海外(雨云海外节点)
- 不备案(海外服务器无需备案)

## 备选方案

| 方案 | 域名 | 备案 | 不选原因 |
|---|---|---|---|
| 国内域名 + 国内服务器 + 备案 | .com | 需要 20 天 | 周期长 |
| 国内域名 + 海外服务器(本方案) | .com | 不需要 | 合理 |
| GitHub Pages 临时域名 | github.io | 不需要 | 不适合 SaaS |

## 影响

- 正面:无需备案,立即上线
- 负面:国内访问延迟 100-300ms
- 风险:国内访问慢影响体验,后续可加 CDN 或迁国内备案
