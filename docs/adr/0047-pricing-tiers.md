# ADR 0047 · 定价档位:免费 + 基础 + VIP

- 状态:accepted
- 日期:2026-07-14
- 层次:商业

## 背景

ADR 0043 定了订阅制,需明确具体价格。

## 决策

| 档位 | 月度 | 终身 | 功能 |
|---|---|---|---|
| 免费版 | ¥0 | - | 开源版自部署 |
| 基础版 | ¥18(限时,正常 ¥30) | 无终身 | SaaS:5 应用 / 1000 卡密 / 基础监控 |
| VIP | ¥68 | ¥128 | SaaS:无限应用 / 无限卡密 / 高级监控 / Web 注入 / 优先支持 |

## 备选方案

| 方案 | 基础 | VIP | 不选原因 |
|---|---|---|---|
| 三档(基础+专业+VIP) | ¥29 | ¥99 / ¥299 | 太复杂 |
| 两档(本方案) | ¥18 | ¥68 / ¥128 | 合理 |
| 单档(仅 VIP) | - | ¥99 | 门槛高 |

## 影响

- 正面:¥18 限时优惠引流,VIP ¥128 终身回笼资金
- 负面:¥18 基础版利润薄(主要靠 VIP)
- 关键:¥18 是限时优惠,正常 ¥30,营造紧迫感

## 关联

- 关联 ADR:0043(订阅制)、0044(发卡网)
