# ADR 0049 · SLA + 退款政策

- 状态:accepted
- 日期:2026-07-14
- 层次:商业

## 背景

SaaS 需明确 SLA 承诺 + 退款政策,写进用户协议。

## 决策

| 项 | 政策 |
|---|---|
| SLA | 可用性 ≥ 99.5%(月度) |
| 故障补偿 | 无 |
| 退款 | 概不退款(数字商品不适用 7 天无理由) |
| 数据备份 | 每日自动备份 + 7 天滚动 |

## 备选方案

| 项 | 方案 | 不选原因 |
|---|---|---|
| SLA | 99% / 99.5%(本方案) / 99.9% | 99.9% 单机做不到 |
| 补偿 | 无(本方案) / 月费 10-50% | 补偿账务复杂 |
| 退款 | 概不退(本方案) / 7 天未使用退 / 按比例退 | 未使用定义模糊 |

## 影响

- 正面:无补偿无退款,运营简单
- 负面:无退款可能影响转化(用户犹豫)
- 法律:消法第 25 条第 4 款,数字商品不适用 7 天无理由,合法
- 关键:用户协议明确"数字商品不适用 7 天无理由"
