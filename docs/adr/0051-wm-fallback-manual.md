# ADR 0051 · WM 发卡不稳定兜底方案

- 状态:accepted
- 日期:2026-07-14
- 层次:商业

## 背景

WM 发卡网还在开发中(ADR 0044),不稳定时用户无法付费。

## 决策

**WM 主 + 手动收款兜底**

- WM 可用时:自动发卡,用户自助兑换
- WM 不可用时:
  1. 用户扫码付款(微信/支付宝个人收款码)+ 备注邮箱
  2. 你手动在后台开通会员
  3. 后续接独角数卡(自建)作为 WM 长期备份

## 备选方案

| 方案 | 备选 | 不选原因 |
|---|---|---|
| 等 WM 稳定再上线 | 阻塞上线 | 时间不可控 |
| WM + 手动兜底(本方案) | 手动收款 | 合理 |
| 多发卡平台 | WM + 独角 + 虎皮椒 | 维护成本高 |
| 直接接支付宝当面付 | 个人不行 | 需企业资质 |

## 影响

- 正面:不阻塞上线,WM 挂时可控
- 负面:手动兜底耗时(MVP 用户少可接受)
- 后续:v2 接独角数卡自建发卡,消除 WM 依赖
