# ADR 0067 · 加固 APK 兼容:MVP 不支持

- 状态:accepted(部分 superseded by 0079,仅梆梆自检场景;其他加固方案仍按本 ADR)
- 日期:2026-07-14
- 层次:注入工具

## 背景

ADR E5 定了"壳 dex 默认 + 自动脱壳可选",需明确 MVP 范围。

## 决策

**MVP 不支持加固 APK,壳 dex 注入放 v2**

- MVP:检测到加固拒绝注入,提示"请提供未加固 APK"
- v2:壳 dex 注入(LSPatch 类方案)+ 自动脱壳(需免责协议)

## 备选方案

| 方案 | 兼容 | 不选原因 |
|---|---|---|
| 不支持(本方案 MVP) | 拒绝 | 合理(MVP) |
| 壳 dex 注入 | hook 业务 dex | 复杂度极高 |
| 自动脱壳 | 解密后注入 | 法律风险(ADR E5) |
| + 运行时 hook | LSPatch | 法律风险 |

## 影响

- 正面:MVP 简单,法律风险低
- 负面:加固 APK 用户需自行脱壳(提供未加固版)
- v2:壳 dex 注入 + 自动脱壳(免责协议)

## 关联

- 关联 ADR:E5(加固兼容)、0029(加固法律风险)
