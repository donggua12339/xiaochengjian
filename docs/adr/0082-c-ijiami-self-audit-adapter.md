# ADR 0082-C · 爱加密自检适配器(V2 评估)

- 状态:draft(暂不开发,V2 评估)
- 日期:2026-07-21
- 决策者:小城笺项目
- 层次:功能 / 安全 / 合规
- 关联 ADR:0078(梆梆适配器,锁 A 扩展)

## 背景

爱加密(Ijiami)是国内主流加固厂商之一,但其 EULA 对商业使用限制较高,V1.5 暂不纳入支持范围。

本 ADR 作为 V2 评估的占位,待 V1.5(梆梆 + 360 + 腾讯乐固)跑通后,视市场反馈及 360 经验再决定是否推进。

## 决策

### 爱加密特征

| 特征 | 值 |
|---|---|
| so 文件 | libexec.so(爱加密标志) |
| Application 类名前缀 | com.ijiami.* |
| EULA | 商业使用限制较高(需律师深度评估) |

### V2 评估条件

以下条件**全部满足**时,本 ADR 从 draft 改为 proposed:

1. V1.5(梆梆 + 360 + 腾讯乐固)已上线,且有 10+ 付费用户使用加固自检功能
2. 有 3+ 用户明确请求爱加密支持
3. 律师完成爱加密 EULA 深度评估,确认风险可控
4. 爱加密 EULA 允许"对加固层做完整性扫描"

### 风险

- 爱加密 EULA 对商业使用限制较高,可能禁止"第三方工具扫描加固层"
- 如果律师评估不通过,本 ADR 改为 rejected,不开发
