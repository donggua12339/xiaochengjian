# 自有 APK 诊断 EULA(梆梆加固自检场景)

> 版本:1.0.0
> 生效日期:2026-07-20
> 适用范围:小城笺"自有 APK 诊断"功能中,启用梆梆加固自检适配器(ADR 0078)的开发者

---

## 1. 声明与承诺

本人在此声明并承诺:

1. **APK 所有权**:本人是待诊断 APK 的合法著作权人或经合法授权的使用者,该 APK 属于本人自有,非第三方所有
2. **不绕过梆梆保护**:本人理解梆梆加固是梆梆公司的技术保护措施,本工具仅对加固层做**完整性扫描**(so/API 文件级哈希),**不脱壳、不反编译、不还原源码**
3. **不用于入侵**:本人承诺不将本工具用于入侵他人系统、绕过他人验证、攻击梆梆公司或其他加固厂商的目的
4. **不扩展厂商**:本人理解本工具仅支持梆梆一家加固厂商(锁 A),360/爱加密/乐固/腾讯乐固等其他厂商明确不支持,本人不要求扩展

## 2. 工具能力边界

本工具(梆梆加固自检适配器)的能力严格限定为:

| 能力 | 允许 | 说明 |
|---|---|---|
| 梆梆 so 文件完整性报告 | ✅ | 列出 so 文件 SHA-256 + 大小 + 加载路径 |
| 梆梆特有 API 调用扫描 | ✅ | 静态规则扫描,不运行 APK |
| APK 签名验证 | ✅ | apksigner verify --print-certs |
| 输出完整性报告(JSON) | ✅ | 仅完整性数据,不含源码 |

| 能力 | 禁止 | 说明 |
|---|---|---|
| 通用脱壳(还原 dex) | ❌ | 违反 ADR 0067(推 v3)|
| 反编译加固 so 还原源码 | ❌ | 违反 ADR 0078 锁 C |
| 动态运行加固 APK | ❌ | 工具不运行 APK |
| 输出反编译源码 / smali | ❌ | 违反 ADR 0078 锁 C |
| 处理非梆梆加固 APK | ❌ | 违反 ADR 0078 锁 A |

## 3. 审计日志

本人理解并同意:

- 本工具的所有操作(诊断 / 回填 / 梆梆自检)都会被记录到 `audit_log_own` 表
- 日志含:开发者 ID / appId / APK hash / 包名 / 签名 hash / 三重校验结果 / EULA 版本 / EULA 接受状态 / IP / User-Agent / 时间戳
- 日志保留 1 年(ADR 0032)
- 日志用于滥用追溯,本人同意小城笺在发现滥用时使用日志举证

## 4. 法律责任

1. 本人理解:若本人违反本 EULA,小城笺有权:
   - 立即终止本人的诊断服务
   - 向第三方(含梆梆公司、执法机关)披露本人的违规行为及审计日志
   - 追究本人的法律责任

2. 本人理解:小城笺不对本工具的准确性做任何明示或暗示的担保,工具输出的完整性报告仅供参考,不构成对 APK 安全性的最终判断

3. 本人理解:本 EULA 受中华人民共和国法律管辖,争议提交小城笺运营方所在地法院管辖

## 5. EULA 变更

- 本 EULA 版本号变化时,本人需重新勾选接受才能继续使用梆梆自检功能
- 本人拒绝新版本时,梆梆自检功能自动禁用,已生成的历史报告保留

## 6. 接受

本人勾选"我已阅读并接受 EULA"即表示接受本 EULA 全部条款,接受状态(含 EULA 版本号 + 时间戳)将记录到 `audit_log_own` 表。
