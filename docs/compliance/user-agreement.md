# 小城笺用户协议

> 本协议是开发者使用小城笺卡密验证系统(以下称"本系统")的法律依据。
> 使用本系统即视为同意本协议全部条款。
>
> **本系统定位**(ADR 0076,2026-07-19):独立开发者的私有应用攻防与遗产维护工具。
> 强调"私有"和"自有",彻底切割与盗版/黑产/通用脱壳/通用去签的关联。

## 1. 定义

- **本系统**:小城笺卡密验证系统,包括开源版与 SaaS 版
- **开发者**:注册本系统账号,使用本系统对其自有应用进行版权保护或自有 APK 诊断的个人或组织
- **最终用户**:开发者分发应用后,使用该应用的用户
- **卡密**:本系统生成的、用于激活开发者应用的授权凭证
- **SDK 集成辅助工具**:本系统提供的 CLI 工具,用于生成 SDK 集成代码模板 + APK 签名/水印(不修改 APK 字节码)
- **自有 APK 诊断**:本系统提供的红方能力,对开发者拥有合法著作权的自有 APK 做只读反编译查看 + 签名信息 + SDK 后门扫描(详见 ADR 0077)

## 2. 授权范围

### 2.1 允许的使用
- ✅ 开发者对其**自行开发**的付费应用做版权保护、防盗版验证
- ✅ 试用/付费转化、设备绑定、卡密销售
- ✅ 使用 SDK 集成辅助工具对开发者**自有著作权**的 APK 进行签名 + 加水印
- ✅ 使用本系统对开发者**合法授权**的 APK 进行签名(需有书面授权)
- ✅ 使用自有 APK 诊断功能,对开发者**拥有合法著作权**的自有 APK 做只读反编译查看(需通过三重校验,详见第 3 节)

### 2.2 禁止的使用
- ❌ 用于规避反作弊系统
- ❌ 用于赌博、诈骗、色情、传销等违法活动
- ❌ 主动提供"绕过其他验证系统"的功能
- ❌ 通用脱壳(对任意 APK 脱壳,FRIDA-DEXDump 类)
- ❌ 通用去签名校验(对任意 APK 去签)
- ❌ 对他人 APK 进行重打包、注入、修改字节码(详见 ADR 0068,v2 已移除此能力)
- ❌ 对本系统进行逆向工程(法律允许的除外)
- ❌ 删除、篡改本系统的水印、版权信息
- ❌ 绕过自有 APK 诊断的三重校验(包名白名单 / 签名比对 / 目录隔离)

## 3. 自有 APK 诊断功能特别声明(ADR 0077)

### 3.1 功能边界

本系统的"自有 APK 诊断"功能是**有限红方能力**,严格限定为:
- 仅对开发者**拥有合法著作权**的自有 APK 做只读反编译查看
- 不提供通用脱壳、通用去签、字节码修改、重打包能力
- 所有诊断操作均为只读,不修改上传的 APK,不输出修改后的 APK

### 3.2 三重校验(强制,法律自证清白的证据)

使用自有 APK 诊断功能前,必须通过以下三重校验,**任一失败即拒绝,不提供跳过开关**:

1. **包名白名单**:APK 包名必须在 admin-web 后台注册(开发者声明为自有应用)
2. **签名 hash 比对**:APK 签名必须与开发者配置的预期签名 hash 匹配(签名密钥即所有权证据)
3. **本地私有目录隔离**:诊断操作在服务器临时目录 `/tmp/audit/<taskId>/` 完成,权限 700,诊断完成后立即删除

### 3.3 法律风险告知

虽然本功能严格限定为自有 APK,但仍需用户保证:
- 对所诊断的 APK 拥有完整著作权,或
- 已获得 APK 著作权人的书面授权
- 承担因使用诊断功能产生的一切法律后果

**注**:本功能不涉及避开技术措施(只读,不修改),不触发《著作权法》第 48 条 / 《计算机软件保护条例》第 24 条 / 刑法 285 条风险(详见 ADR 0076 法律评估)。

### 3.4 平台权利
- 平台保留配合司法调查的权利
- 平台保留在收到合法投诉时暂停或终止用户账号的权利
- 平台保留修改或下线诊断功能的权利
- 所有诊断操作记录审计日志(保留 1 年,详见 ADR 0077)

## 4. 加固 APK 兼容性声明

### 4.1 MVP 不支持加固 APK(ADR 0067)

本系统 v2 的 SDK 集成辅助工具**不支持**对加固 APK(360/腾讯乐固/爱加密/梆梆等)的处理:
- 检测到 APK 加固特征时,工具拒绝签名
- 提示开发者提供未加固版本的 APK
- 加固 APK 兼容(壳 dex 注入 / 自动脱壳)推迟到 v3,且需免责协议

### 4.2 法律风险告知(未来 v3 脱壳功能)

v3 若实现自动脱壳功能,涉及绕过技术保护措施,可能违反:
- 《中华人民共和国著作权法》第 48 条
- 《计算机软件保护条例》第 24 条

使用脱壳功能的用户需保证:
1. 对所脱壳的 APK 拥有著作权,或
2. 已获得 APK 著作权人的书面授权
3. 承担因使用脱壳功能产生的一切法律后果

### 4.3 平台权利
- 平台保留配合司法调查的权利
- 平台保留在收到合法投诉时暂停或终止用户账号的权利
- 平台保留修改或下线相关功能的权利

## 4. 数据处理

### 4.1 数据所有权
- 开发者账号数据归开发者所有
- 卡密数据归开发者所有
- 最终用户设备数据(机器码等)归开发者所有,平台代为保管

### 4.2 数据使用
- 平台不向第三方出售开发者数据
- 平台仅在有合法依据时向司法机关提供数据
- 平台使用聚合数据(不可识别个人)用于运营分析

### 4.3 数据安全
- 平台采取合理技术措施保护数据安全
- 卡密明文不存储,仅存哈希
- 数据传输全程加密

### 4.4 数据删除
- 开发者可随时申请删除账号及所有数据
- 平台收到合法删除请求后 30 日内完成
- 法律要求保留的数据除外

## 5. 服务水平

### 5.1 SaaS 版
- 平台承诺 SaaS 版可用性 ≥ 99.5%
- 维护公告提前 24 小时通知
- 故障响应时间 ≤ 1 小时(工作日)

### 5.2 开源版
- 开源版由开发者自行部署,平台不提供可用性承诺
- 平台提供文档与社区支持
